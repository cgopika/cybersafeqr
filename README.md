# 🛡️ Cyber Safe QR

> An interactive Cyber Awareness Website developed by a four-member student team during the **Cyber Security & Investigation Internship** at **Cyber Crime Police Station, Palakkad**.

---

# 🌐 Live Website

https://cybersafeqr.onrender.com

---

# 📜 Project Overview

Cyber Safe QR is an interactive cyber awareness platform that helps the public learn about different cyber crimes through simulations, awareness activities, legal information, and safety tips.

The website is designed to make cyber security education simple, interactive, and engaging.

---

# 🎯 Objectives

- Spread cyber awareness
- Educate people about online scams
- Teach safe online practices
- Explain cyber laws
- Promote reporting of cyber crimes

---

# ✨ Features

- Financial Crime Awareness
- Social Media Crime Awareness
- Hacking Attack Awareness
- Miscellaneous Cyber Crimes
- Interactive Simulations
- Cyber Law Information
- Mobile Friendly Design
- QR Code Access
